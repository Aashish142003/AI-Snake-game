# AI Snake Game

This repository contains an AI-powered Snake game with both a manual playable mode and an automated AI solver. 

## 🚀 Features
- **Manual Mode**: Play the game yourself using Arrow keys or WASD.
- **AI Solvers**: includes Greedy and Hamiltonian Cycle solvers.
- **Benchmark Mode**: Run the AI in the terminal to measure performance without a GUI.

## 🛠️ Installation
The game requires `pygame`. You can install it using:
```bash
pip install pygame
```

## 🎮 How to Run

### Manual Playable Game
```bash
python3 Manual_SnakeGame.py
```
- **Controls**: Arrow Keys or WASD.
- **Start**: Press `SPACE`.
- **Retry**: Press `R` after game over.

### AI Snake Game (Benchmark)
```bash
python3 AI_SnakeGame.py
```
*Note: Currently configured for Benchmark mode to run without a GUI.*
